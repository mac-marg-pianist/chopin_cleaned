이 파일들은 Yamaha-e-competition에 공개된 연주 곡 중, XML 파일을 구할 수 있는 연주들에 한해 곡별로 악보와 함께 정리한 것들 입니다.


구성 요소:

악보
xml.xml  - 인터넷에서 구한 악보 파일 입니다.
xml.mxl  - 인터넷에서 구한 악보 파일의 압축본 이며, unzip 해서 xml 파일을 얻을 수 있습니다. xml에 대한 추가적인 정보가 있을지 몰라 임시로 저장했습니다.
midi.mid - xml.xml 파일을 musescore를 이용해서 midi 형식으로 바꾼 파일입니다.

연주
*.mid    - Yamaha competition에서 다운 받은 파일들이며, yamaha e-competition에서 파일을 다운 받았을때 이름을 그대로 쓰고 있습니다.
*XP.mid  - XP(yamaha에서 개발한 고해상도 midi형식) 으로 된 파일이며, 같은 이름으로 된 XP가 없는 미디파일과 내용이 동일합니다. XP파일이 없는 경우가 있습니다.

정렬결과
*.txt    - Nakamura의 midi-to-midi alignment 알고리즘을 통해 매칭된 결과를 저장한 파일입니다. (https://midialignment.github.io/demo.html)
           자세한 내용은 Nakamura의 manual을 참고하세요. 매칭 실패로 인해 매칭되지 않은 파일도 존재합니다.
